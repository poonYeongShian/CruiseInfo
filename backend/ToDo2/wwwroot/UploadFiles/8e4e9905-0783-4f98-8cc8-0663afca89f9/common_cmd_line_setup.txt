setup reference: https://blog.talllkai.com/ASPNETCore/2021/03/09/APICRUD
1. Add database file into Server Explorer tab
2. Run command--> Scaffold-DbContext "Data Source=(LocalDB)\MSSQLLocalDB;AttachDbFilename=C:\Users\User\Desktop\ASPNET_learn\prac2netDTOS\data\Todo4.mdf;Integrated Security=True;Connect Timeout=30" Microsoft.EntityFrameworkCore.SqlServer -OutputDir Models -Force -CoNtext TodoContext
3. Put this into appsetting.json
  "ConnectionStrings": {
    "TodoDatabase": "Data Source=(LocalDB)\\MSSQLLocalDB;AttachDbFilename=D:\\Todo.mdf;Integrated Security=True;Connect Timeout=30"
  },
4. Add dependency in startup.cs
public void ConfigureServices(IServiceCollection services)
{
    services.AddDbContext<TodoContext>(options =>
        options.UseSqlServer(Configuration.GetConnectionString("TodoDatabase")));
}
5. In TodoContext.cs, delete OnConfiguring
